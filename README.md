# DOCKERIZED_NVCHAD

![](ZZZ/ZZZ.jpg)

* Do you love `NEOVIM` ?
* Do you love `NV_CHAD` ?
* Do you love `DOCKER` ?

* If yes, You are at the right place.

* Here is a persistent configuration for `NV_CHAD` for docker container
* You can run `NEOVIM` with `NV_CHAD` configuration all in docker with all data being persistent in volumes.
* Apart from testing in docker container, now you are able to even save and completely edit all your projects, poems and many more... all in docker containers

# USAGE
* Clone this repo
```bash
git clone https://github.com/P-Y-R-O-B-O-T/DOCKERIZED_NVCHAD && cd DOCKERIZED_NVCHAD
```

* Setup things
```bash
chmod +x install.sh
./install.sh
```

* Run your very own dockerised `NV_CHAD`
```bash
sudo doc_nvchad
```
* Use `NV_CHAD` using ty issuing `nvim` command

* Whenever you need to again use nvim just use the last command
